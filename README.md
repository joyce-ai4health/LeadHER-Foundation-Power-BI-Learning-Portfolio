# LeadHER Foundation – Power BI Learning Portfolio

This repository documents my learning journey in **Power BI** through the **LeadHER Foundation Data Analytics Program**.
The program is an **8-week training** designed to build practical skills in **data preparation, data modeling, visualization, dashboard design, and business intelligence reporting**.

Each week includes lessons, practical exercises, and assessments that demonstrate my growing proficiency in **Power BI and data analytics**.

---

# Course Structure

## Week 1 – Getting Started with Power BI

Topics covered:

* Welcome Address
* Course Content Overview
* Introducing Power BI
* Installing Power BI
* The Power BI Interface
* Importing a CSV File
* Importing Data from the Web
* Creating a Simple Bar Chart Visualization
* Saving a Power BI File
* Real-World Use Cases of Power BI
* Week 1 Assessment
